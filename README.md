# Morphic Grid Project

## Overview

The Morphic Grid Project is a security-based application that generates unique visual patterns using the behavior of ferrofluids. By leveraging the dynamic and unpredictable nature of ferrofluids under magnetic fields, the system creates distinct morphic grid patterns that can be used for authentication and security applications.

## Features

- Generate unique ferrofluid-based patterns
- Secure pattern generation for authentication
- Interactive web interface
- Database integration for storing generated patterns
- Easy-to-use and responsive design

## Installation

1. Clone the repository

```bash
git clone https://github.com/priyanka28092006-ship-it/ep-project.git
```

2. Navigate to the project folder

```bash
cd ep-project
```

3. Install the required dependencies

```bash
pip install flask
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```

## Working

The application simulates morphic grid generation using ferrofluids. The generated patterns are unique due to the complex behavior of ferrofluids when influenced by magnetic fields. These patterns can be stored and verified for security and authentication purposes.
